Buffer(буфер)

Буфер - устройство временного хранения, в общем случае используемое для компенсации разницы скоростей или тактирования при обмене данными между устройствами. Буферизация также используется для подавления дрожи.

A buffer is a temporary storage device, generally used to compensate for speed differences or clocking when exchanging data between devices. Buffering is also used to suppress tremors.
