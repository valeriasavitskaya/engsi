Encapsulation(инкапсуляция)

Инкапсуляция - метод, используемый многоуровневыми протоколами, в которых уровни добавляют заголовки в модуль данных протокола из вышележащего. В терминах Internet - пакет содержит заголовок физического уровня, за которым следует заголовок сетевого уровня, а за ним заголовок транспортного уровня, за которым располагаются данные прикладных протоколов.

Encapsulation is a method used by layered protocols in which layers add headers to the protocol data module from the overlying one. In Internet terms, a packet contains a physical layer header, followed by a network layer header, followed by a transport layer header, followed by application protocol data.
