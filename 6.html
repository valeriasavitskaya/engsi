Attribute(атрибут)

Атрибут - форма информации, обеспечиваемая Службой каталогов X.500. Базовая информация о каталогах состоит из записей, каждая из которых содержит один или несколько атрибутов. Каждый атрибут содержит тип идентификатора и одно или несколько значений. Каждая операция чтения каталога может отыскивать некоторые (или все) атрибуты в заданной записи.

An attribute is a form of information provided by the X.500 Directory Service. Basic directory information consists of records, each of which contains one or more attributes. Each attribute contains an identifier type and one or more values. Each directory read operation can search for some (or all) attributes in the specified record.
